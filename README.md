# 邮游世界 - 个性化旅游推荐系统

## 项目简介

邮游世界是一个基于大语言模型和自研算法的个性化旅游规划系统，为用户提供智能行程规划、景点推荐、路线规划等功能。

## 系统功能

### 1. 旅游推荐
- 基于热度、评价和个人兴趣推荐目的地
- 支持名称、类别、关键字查询

### 2. 路线规划
- 最优路径：规划起点到单个或多个目标点的最优线路
- 多种策略：最短距离、最短时间、步行/自行车/电瓶车
- 可视化：提供地图展示和路径输出的图形界面
- 室内导航

### 3. 场所查询
- 查找景区/校园内附近的设施（超市、卫生间）
- 支持按类别过滤和距离排序

### 4. 旅游日记
- 记录与管理：支持文字、图片、视频记录
- 检索与评价：支持全文检索、精确查询及基于热度/评价的推荐

### 5. 美食推荐
- 按热度、评价、距离和菜系推荐美食
- 支持模糊查询

### 6. AI助手
- 使用AIGC算法生成旅游动画
- 智能解析小红书行程链接

## 技术栈

| 组件 | 技术 |
|------|------|
| 后端 | FastAPI + Python |
| 前端 | Vue 3 + Element Plus |
| 数据库 | SQLite |
| AI | MiniMax API |

## 核心算法

| 算法 | 说明 |
|------|------|
| Dijkstra | 最短路径算法 |
| TSP | 旅行商问题求解 |
| TopK | 部分排序推荐 |
| 模糊查找 | 支持拼音首字母、通配符 |
| Gzip | 无损压缩 |

## 数据规模

- 景区数量: ≥200个
- 道路边数: ≥200条
- 服务设施: ≥10种
- 建筑数量: ≥20个/景区

## 快速启动

### Windows
双击 `启动.bat` 即可启动前后端

### 手动启动
```bash
# 后端
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --port 8000

# 前端
cd frontend
npm install
npm run dev
```
